# 桌面宠物
一个基于 PyQt5 的 AI 桌面宠物聊天程序，支持 DeepSeek 和通义千问大模型。有聊天界面，语音播报，可自定义人设与头像。

## 功能特点
- 聊天界面
- AI 智能对话（支持 DeepSeek / 通义千问）
- 语音播报（支持多种音色切换）
- 桌面宠物动画（呼吸、眨眼、边缘巡逻）
- 可自定义多人设，随时切换
- 单击摸头、双击打开聊天

## 使用方法
1. 下载最新版本的压缩包（dist.zip）
2. 解压到任意文件夹
3. 双击 `桌宠.exe` 运行
4. 首次运行需要配置 API Key（DeepSeek 或通义千问）

## 需要的 API Key
- **DeepSeek**：前往 [platform.deepseek.com](https://platform.deepseek.com) 获取
- **通义千问**：前往 [dashscope.aliyun.com](https://dashscope.aliyun.com) 获取
